Use Command Line Interface (CLI)

If you're comfortable using the CLI, you can perform tasks directly through the terminal interface.

CLI Commands:

  • upload: Upload files
  • analyze --type [descriptive|linear|tree]: Perform analysis
  • report: Generate report
